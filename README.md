## How to use

1. Edit `index.html` to add or update post links.
2. Create post files inside `posts/` (example: `posts/lactf-1.html`).
3. Keep styles in `home.css` and assets in `/assets`.

## Post format

Use this structure in each post:

- Same header block (`left` section with logo and name).
- Main content in `<section id="right">`.
- Post title in `<h2>`.
- Optional table of contents using `<nav class="toc">`.
- Content inside `<article class="member writeup"><div class="info">...</div></article>`.
- Use `<pre><code>` for code snippets and commands.
